# Atividade ESP32 - LARC

Este repositório contém os códigos e simulações desenvolvidas para a atividade da LARC sobre a plataforma ESP32.

---

### 1. Comunicação Serial

* **Descrição:** Demonstração de envio e recebimento de dados (como comandos) entre o computador e o ESP32 através do Monitor Serial (Ex.: controlar um LED).
* **Código-Fonte:** [**`Comunicacao_serial.ino`**](./Comunicacao_serial.ino)
* **Simulação Wokwi:** [![Wokwi](https://img.shields.io/badge/Wokwi-Simulação_Serial-blue?logo=wokwi&logoColor=white)](https://wokwi.com/projects/447615334056001537)

### 2. Leitura Analógica

* **Descrição:** Exemplo de como o ESP32 pode ler valores de sensores analógicos (como um potenciômetro), convertendo a tensão de entrada em um valor digital (de 0 a 4095).
* **Código-Fonte:** [**`Leitura_analogica.ino`**](./Leitura_analogica.ino)
* **Simulação Wokwi:** [![Wokwi](https://img.shields.io/badge/Wokwi-Leitura_Analógica-blue?logo=wokwi&logoColor=white)](https://wokwi.com/projects/447614617003951105)

---
1.  Clique no link de "Código-Fonte" para ver o script `.ino` usado.
2.  Clique no "badge" da simulação para abrir o projeto no Wokwi.
3.  No Wokwi, clique no botão de **Play** verde para iniciar a simulação.
